# TURING-CAMERA-TIMELAPSE

Service for recording timelapse photo files from camera feed


## Install requirements

- Requires numpy:
  - On osx:
      pip install numpy
  - On rpi:
      sudo apt-get install python-numpy

- Requires ffmeg:
  - On osx:
      brew install ffmpeg
  - On rpi:
      sudo apt-get update
      sudo apt-get install ffmpeg
